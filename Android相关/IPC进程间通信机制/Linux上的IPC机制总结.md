# Linux上IPC机制总结

Linux上主要采用以下集中集中IPC机制进行进程间的通信

1. **管道 （PIPE）**
2. **命名管道（FIFO）**
3. **信号 （signal）**
4. **消息队列（Message queues）**
5. **信号量（Semaphore）**
6. **共享内存（Share Memory）**
7. **内存映射（Memory Map）**
8. **套接字**
